## 1. プロジェクト概要

**Sales Pilot AI** は、新規開拓営業を効率化する社内ツールです。

```
 企業を探す　→　メールを送る　→　返信・商談を管理する　→　成約/失注を記録する
```

これまで営業担当が手作業で行っていた「企業リスト作成」「メール送信」「進捗管理」を、1つのツール（CLI）で完結できるようにしています。

| 項目 | 内容 |
| --- | --- |
| 目的 | 新規開拓営業の作業時間を削減し、営業担当を商談・クロージングに集中させる |
| 管理方法 | 使い慣れた **Google スプレッドシート** をそのままデータベースとして利用 |
| 送信手段 | **Gmail** から自動送信・自動記録 |
| 利用者 | 営業担当者（日次利用を想定） |

## 4. 現在利用できる機能

| 機能 | できること |
| --- | --- |
| 企業検索 | 業種・地域を指定して候補企業を自動収集 |
| レビュー付き登録 | 検索結果を確認しながら1件ずつ登録するか選べる |
| メール送信 | 対象企業へ一括送信（誰に・何を送るか事前確認、送りたくない相手は今回だけ除外可能） |

| 返信管理 | 返信の有無を自動検知して記録 |
| 進捗更新 | 商談日・成約・失注・メモを記録 |
| 営業レポート | 送信数・返信率・成約数などを集計表示 |
| フォローアップ | 今日連絡すべき相手を優先順位順に表示 |

## 5. CLIコマンド一覧

営業担当・エンジニアが実行する操作の一覧です（参考情報）。

| コマンド | 何をするか |
| --- | --- |
| `sales-pilot find` | 企業を検索してリストに追加する |
| `sales-pilot send` | メールを送信する |
| `sales-pilot update` | 商談・成約・失注などの進捗を更新する |
| `sales-pilot report` | 営業状況のレポートを表示する |
| `sales-pilot follow-up` | 今日のフォロー対象を表示する |
| `sales-pilot template` | メールテンプレートを管理する |

## 6. システム構成図

```
        営業担当者
            │  コマンドを実行
            ▼
   ┌──────────────────┐
   │   Sales Pilot AI  │   ← 操作の窓口（CLI）
   └────────┬─────────┘
            │
   ┌────────┼─────────────────┐
   ▼                          ▼
┌───────────────┐     ┌───────────────┐
│ Google スプレッド │     │   Gmail        │
│ シート           │     │（メール送信・受信）│
│（企業リスト・履歴・│     │                │
│  ダッシュボード） │     └───────────────┘
└───────────────┘
            ▲
            │ 文面を選択
   ┌──────────────────┐
   │  メールテンプレート │
   └──────────────────┘
```

普段使っている **Google スプレッドシート** がそのままデータの中心（正）になっているため、ツールを使わずシートを直接見ても内容を確認できます。
